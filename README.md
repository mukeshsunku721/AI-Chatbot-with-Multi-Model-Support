# AI-Chatbot-with-Multi-Model-Support
Built an AI chatbot using Groq API and LLaMA models to generate intelligent conversational responses. Implemented multi-model switching and chat history management with a responsive ChatGPT-style web interface.
# AI Chatbot Web Application

An AI-powered chatbot built using Flask and Groq API with LLaMA models.

## Features

- Multi-chat conversation system
- Chat history sidebar
- Rename and delete chats
- Dark / Light mode
- Model selection
- Typing animation
- Markdown rendering

## Tech Stack

- Python (Flask)
- HTML, CSS, JavaScript
- Bootstrap
- Groq API (LLaMA models)

## Installation

```bash
git clone https://github.com/yourusername/ai-chatbot.git
cd ai-chatbot
```
