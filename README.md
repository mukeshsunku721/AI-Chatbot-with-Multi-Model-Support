# 🤖 AI-Chatbot-with-Multi-Model-Support

An **AI-powered chatbot web application** built using **Flask and Groq LLM APIs**.  
The project provides a **ChatGPT-like interface** with chat history, model selection, dark/light theme, and multiple conversation management features.

This project demonstrates **full-stack development and AI API integration**, making it suitable for AI/ML, Data Science, and Software Engineering portfolios.

---

# 🚀 Features

- 💬 Interactive chatbot interface
- 🧠 AI responses using **Groq LLM models**
- 🗂 Multiple chat sessions with sidebar history
- ✏️ Rename chats manually
- 🗑 Delete chats
- 🌙 Dark / Light mode toggle
- 🔄 Model selection (multiple LLM models)
- ⌨️ Send message using **Enter key**
- ⏳ Typing animation for AI responses
- 📝 Markdown rendering for formatted responses
- 💾 Chat history stored in **local storage**

# ⚙️ Installation

### 1️⃣ Clone the repository
``` bash
git clone https://github.com/mukeshsunku721/AI-Chatbot-with-Multi-Model-Support.git

cd AI-Chatbot-with-Multi-Model-Support
```

### 2️⃣ Install dependencies
``` bash
pip install -r requirements.txt
```

### 3️⃣ Create `.env` file

Create a `.env` file in the root directory:
``` bash
GROQ_API_KEY=your_api_key_here
```

### 4️⃣ Run the application
```bash
python app_offline.py
```

Open the application in your browser:
``` bash
http://127.0.0.1:5000
```
